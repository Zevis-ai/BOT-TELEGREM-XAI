# 🤖 גרוק-בוט טלגרם
> הבוט הכי מגניב בטלגרם שמדבר באמצעות Grok!

## 🌟 מה הבוט יודע לעשות?
- 💬 מדבר איתך בעברית (או בכל שפה אחרת!)
- 🧠 משתמש ב-Grok AI בשביל לתת תשובות חכמות
- 🚀 מהיר וקל לשימוש

## 📋 דרישות מערכת
- Python 3.8+
- חשבון טלגרם
- טוקן של Grok AI
- הרבה מצב רוח טוב 😊

## 🔧 התקנה

### שלב 1: שכפול הפרויקט
העתק והדבק בטרמינל:
git clone https://github.com/YOUR_USERNAME/grok-telegram-bot.git

### שלב 2: התקנת חבילות
העתק והדבק בטרמינל:
pip install -r requirements.txt

### שלב 3: הגדרת משתני סביבה
צור קובץ .env והוסף את השורות הבאות:
TELEGRAM_TOKEN=your_telegram_token
XAI_API_KEY=your_xai_api_key

## 🚀 הפעלה
הרץ את הבוט בטרמינל:
python bot.py
